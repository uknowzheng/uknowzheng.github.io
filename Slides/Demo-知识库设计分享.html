知识管理核心理念

解决问题

  • 信息碎片化，难以查找
  • 知识无法复用
  • 笔记之间缺乏关联

核心思路

双链 + 标签 + 结构化 = 知识网络

MOC 索引设计

什么是 MOC

MOC = Map of Content 内容地图

作用

  • 总索引 → 各领域 MOC
  • 每个领域有自己的索引页
  • 通过双链连接相关笔记
  • 构建知识网络
graph LR
    A[总索引] --> B[Learning MOC]
    A --> C[Work MOC]
    A --> D[Life MOC]
    B --> E[AI学习]
    B --> F[编程技术]
    C --> G[项目管理]
    C --> H[文档]

## 总结

### 关键要点

1. **结构清晰** - 目录规范统一
2. **双链优先** - 用链接而非文件夹
3. **标签补充分类** - 多维度检索
4. **模板提效** - 新建笔记自动化
